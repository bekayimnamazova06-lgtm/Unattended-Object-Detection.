// Capabilities

Features

🔍

Real-Time Detection

YOLOv8 nano runs on every frame — detecting backpacks, handbags, and suitcases with confidence scoring.

🧠

Centroid Tracking

Each object gets a unique ID tracked across frames using nearest-centroid matching. New objects register automatically.

⏱

Per-Object Timer

Every tracked object has its own stationary timer. Movement resets the clock. Stillness counts down to the alert.

🚨

Visual Warnings

Unattended objects get a red bounding box and "UNATTENDED!" label. A full-width banner appears at the top.

🎭

Configurable

Adjust the alert timer, movement tolerance, and confidence threshold with simple constants in main.py.

📹

Webcam or Video

Works with a live webcam or any video file — change one line in the source configuration.

// Architecture

How It Works

1 · Video Frame Capture

OpenCV reads each frame from webcam or video file at native FPS.

2 · YOLOv8 Inference

Frame passed to YOLOv8. Results filtered to backpack / handbag / suitcase with confidence ≥ 0.4.

3 · Centroid Extraction

Each bounding box center is computed and fed to the ObjectTracker.

4 · Track Matching

New centroids matched to existing tracks by Euclidean distance. Unmatched detections spawn new tracks.

5 · Stationary Timer

Object moves >30px → reset timer. Stays still → timer accumulates. At 5s → UNATTENDED.

6 · Annotated Output

Frame drawn with bounding boxes, labels, timers, and warning overlays via cv2.imshow().
